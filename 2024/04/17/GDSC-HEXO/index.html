Hexo 使用 Markdown 來撰寫文章。在 source/_posts 目錄中，你可以找到示例文章 hello-world.md。使用 Markdown 語法編寫你的文章。
當然，我可以提供一個更詳細的 Markdown 編寫教學，特別針對從未使用過 Markdown 的學生。讓我們來深入了解 Markdown 的基礎知識：

Markdown 是什麼？

Markdown 是一種輕量級的標記語言，用於編寫格式化的文本，以便於轉換為HTML等其他格式。它通常用於撰寫網頁內容、Readme 文件、網誌文章等。

基本語法

標題

文字格式

你可以使用下列方式來調整文字的格式：

  • 粗體: 使用 ** 或 __ 包圍文字來顯示粗體，例如：**這是粗體文字**。
  • 斜體: 使用 * 或 _ 包圍文字來顯示斜體，例如：*這是斜體文字*。
  • 刪除線: 使用 ~~ 包圍文字來顯示刪除線，例如：~~這是刪除線文字~~。

列表

鏈接和圖片

你可以使用 [顯示文字](連結) 來添加超鏈接，例如：[OpenAI](https://openai.com)。

要添加圖片，使用 ![圖片描述](圖片連結)，例如：![OpenAI Logo](https://openai.com/assets/logo-d5c8e8d92bca0b3ad88b6ce56167b88f1fc66b1765700722ff0b20a08d0d1ee0.png)。
